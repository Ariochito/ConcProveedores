# Conciliador de Proveedores CFDI

Herramienta web para analizar y conciliar CFDI de proveedores recibidos en formato XML SAT. Permite detectar diferencias, duplicados y generar reportes claros para auditoría y control contable.

## 🚀 Características
- Importación de CFDI XML recibidos.
- Cruce automático con registros contables.
- Reportes de diferencias, duplicados y pendientes.
- Exportación de resultados en Excel/CSV.
- Interfaz intuitiva con filtros por fechas, montos y estatus.

## 📦 Instalación
Este proyecto se ejecuta directamente en el navegador gracias a GitHub Pages.  
No requiere instalación local ni dependencias adicionales.

## 🌐 Acceso
Puedes acceder al proyecto en línea desde:  
👉 [Conciliador de Proveedores](https://ariochito.github.io/ConcProveedores/)

## 🛠 Uso
1. Sube tus archivos XML de proveedores.  
2. Aplica filtros por fecha, monto o estatus.  
3. Revisa los reportes generados.  
4. Exporta los resultados para auditoría o conciliación contable.
